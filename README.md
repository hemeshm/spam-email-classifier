# Spam Email Classifier

This project detects spam and non-spam emails using machine learning and natural language processing techniques.

## Overview
The system cleans and processes email text, then classifies each email as spam or ham using a trained machine learning model. It is designed to improve inbox security and reduce manual filtering effort.

## Features
- Email text preprocessing
- Spam vs ham classification
- Model training on labeled data
- Prediction on new email text
- Performance evaluation
- Confusion matrix and sample prediction output

## Tech Stack
- Python
- NLTK
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Joblib
